# rock_paper_scissor

The goal for this project is to make a rock-paper-scissor game in the console.

i will aproach the problem with the following steps.

- decompose the project into sub-problems.
- write pseudocode with comments.
- translate psuedocode to javascript.
- test for edge cases.
